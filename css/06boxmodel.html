Box Model

el box model es el algoritmo a traves del cual el navegador dibuja las cajas en la pantalla, solo se aplica a elementos de bloque, cuenta con tres elementos

  • Contenido--> content-box
  • separacion entre el contenido y el borde--> padding-box
  • Border--> border-box
  • margin--> propiedad externa,separacion de la caja abyacente
